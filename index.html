What I'm working on

Nathan Barker, Andrew Duncan and myself have submitted a paper which describes an algorithm to solve the power conjugacy problem in the Higman-Thompson groups $G_{n,r}$. This paper builds on Higman's work on the conjugacy problem.

As part of my contribution toward the paper, I have implemented the nitty-gritty details of this algorithm in a Python package called thompson. The package consists of two layers: a lower layer for working with the algebras $V_{n,r}$, and a higher layer for working with automorphisms of these algebras. For more information, see the GitHub repository, the project documentation, or the paper.

At the same time, I'm also taking Kay Magaard's MAGIC course on Representation Theory of Groups in semester two.

Papers

The power conjugacy problem in Higman-Thompson groups, with Nathan Barker and Andrew Duncan. Submitted (March 2015).

An introduction to the universal algebra approach to Higman-Thompson groups (including Thompson's group $V$) is given, following a series of lectures by Graham Higman in 1973. The algorithm for the conjugacy problem outlined in Higman's lectures is shown to be incomplete, and a revised and complete version of the algorithm is written out explicitly. An algorithm for the power conjugacy problem in these groups is constructed based on the conjugacy algorithm. Python implementations of these algorithms can be found online.
